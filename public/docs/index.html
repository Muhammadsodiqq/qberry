Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost

Authenticating requests

This API is not authenticated.

Api for roles

Get all roles

Example request:
curl --request GET \
    --get "http://localhost/api/role/get" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/role/get"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Api for users

Login User

Example request:
curl --request POST \
    "http://localhost/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"iqzc\",
    \"password\": \"ps\"
}"

const url = new URL(
    "http://localhost/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "iqzc",
    "password": "ps"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/login

Body Parameters

email  string  

Must be a valid email address. Must not be greater than 40 characters.

password  string  

Must be at least 4 characters.

Create User

Example request:
curl --request POST \
    "http://localhost/api/user/create" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"fzwmlqejhynoxxktalvzwvge\",
    \"email\": \"mmjivmudkfvkbcmkbamupqooxpfencuop\",
    \"role_id\": 3

    "http://localhost/api/user/create"

const headers = {
    "Content-Type": "application/json",

Request      

POST api/user/create

Body Parameters

name  string  

The name of the user. Must not be greater than 255 characters.

email  string  

The email of the user. Must be a valid email address. Must not be greater than 40 characters.

role_id  integer  

The role id of the user.
